Research

AI systems for real cyber defense.

My research connects machine learning, autonomous agents, cloud infrastructure, and cyber operations into systems that can be measured, audited, and improved.

Autonomous Agent Alliances

Coordinated defense across specialized agents

Agent alliances can route artifacts, delegate analysis, combine evidence, and escalate decisions for human review. My AICCC work explores this model for cyber defense using PCAP routing, PE analysis, network anomaly detection, opcode modeling, feedback loops, and audit records.

Machine Learning IDS

Engineered features, XGBoost, and real-time Zeek analysis

My UCC 2024 work studies practical IDS performance through reproducible preprocessing, hyperparameter optimization, model evaluation, and live network-log scoring. The focus is not just model accuracy; it is operational viability.

Malware and Opcode Modeling

Classical ML for executable behavior

Opcode-sequence analysis remains valuable for malware detection when paired with structured preprocessing and interpretable scoring. I use HMM-style modeling as one component in a broader autonomous defense pipeline.

Cloud Security

Security engineering for distributed systems

Cloud environments demand controls that account for identity, telemetry, runtime behavior, network exposure, and infrastructure drift. My engineering work emphasizes practical hardening and defensible architectures.
